

## Installation

To get started, follow these steps:

1. **Clone the repository**:
    ```bash
    [git clone https://github.com/Codevineet/Arc-Tech-Labs-Projects.git](https://github.com/Codevineet/YoungLabs_Assignment.git)
    cd YoungLabs_Assignment
    ```

2. **Backend**:
    ```bash
    cd backend
    npm install
    npm run dev
    ```

3. **Frontend**:
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
